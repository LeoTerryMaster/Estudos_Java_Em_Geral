Segunda aula revisão de html

Estrutura básica com tags

Parágrafo para escrtia em html tag (br) para quebra de linha
ou tag (br /> ).

tag html (abre e fecha a tag ), tag principal onde tudo começa
tag head (abre e fecha tag ), teg que significa cabeça, e vai avima do corpo
e muitas vezes vais no corpo para estruturar o corpo de um modo diferente.
teg body (abre e fecha ), é o corpo do site onde fica toda a estrutura do site.
