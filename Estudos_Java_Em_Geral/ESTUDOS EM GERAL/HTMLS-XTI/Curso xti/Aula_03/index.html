Estudos de htmls TAGS

Tags é tudo aquilo que vem entre menor que < e maoir que >

para escrever uma tag precisamos abrir e fechar com exceções de algumas tags
que não precisam abrir e fechar

Para escrever uma html precisamos saber que temos certas tags para escrever corretamente
tendo elas html com < > que abre e fecha < menor que html maior que >para abrir a tag html,
para fechar a tag precisamos das seguintes descrição < menor que seguida de barra invertida /html maior que> para finalizar,


Logo temos a tag head que significa a cabeça do nosso site com a mesma escrita que temos no exemplo acima,
temos dentro da tag head a tag title para descrever o título do site.

Logo abaixo da tag head temos o corpo do site que tem a tag body, tudo que esta dentro desse corpo pertence a tag body,
que por sua vez descreve todo o site em si e seu comportamento.
